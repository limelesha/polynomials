# polynomials
Python module for computations with polynomial functions

# Usage Example
```python
>>> a = Poly(-3.495, 1.35, 0.775, -0.425, 0.05)
>>> print(a)
0.05x⁴-0.425x³+0.775x²+1.35x-3.495
>>> a.roots()
[-1.8588312587810811, 2.41517538287795, 3.5185951699760416, 4.42506070592709]
>>> a.minima()
[-0.574745271964034, 4.049682691461983]
>>> a.integral(1, 3)
-0.9533333333333305
>>> a.derivative()
Poly(1.35, 1.55, -1.275, 0.2)
>>> print(a * 15 + 4.5)
0.75x⁴-6.375x³+11.625x²+20.25x-47.925000000000004
>>> a * Poly.with_roots(4, 5)
Poly(-69.9, 58.455, -0.14499999999999957, -14.125, 5.6, -0.875, 0.05)
>>> Poly.with_roots(-9, -7, -2.44, 0.005).roots()
[-9.000000000000028, -6.999999999999974, -2.440000000000002, 0.005]
>>>
```
